# AI Resume Analyzer PRO

AI-powered ATS resume optimization platform built with Next.js and Tailwind CSS.

## Features
- ATS Score Analysis
- Resume vs Job Description Matching
- Missing Skills Detection
- Modern SaaS UI
- AI-style Resume Feedback
- Responsive Design

## Tech Stack
- Next.js
- React.js
- Tailwind CSS
- TypeScript
